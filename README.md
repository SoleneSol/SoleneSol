## Hi !

I'm Sol, a french music addict, learning how to code !

Currently, i'm working on a full stack app for secret santa :
Draw Give will be a Christmas gift exchange web app. 
Users join multiple family groups, keep year-round personal wishlists, and take part in randomized gift draws that assign each member a recipient to shop for. 
The stack is React on the frontend and Node.js/Express/Sequelize on the backend, with gifts tied to a GiftList join model (per user, per family) to avoid cross-family duplication, and an invitation-code system for joining families. 
The UI follows a dark, elegant design system (#30332E background, golden #FDC149 accents, Playfair Display/Cormorant Garamond fonts) with pages for registration, login, and a two-column dashboard covering profile, families, gifts, and draws. 

Stay tuned !
